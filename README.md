# To Do List
Aplicación web desarrollada con **React**, **TypeScript** y **Tailwind CSS** para gestionar tareas de forma sencilla e intuitiva. El proyecto fue creado con el objetivo de profundizar en el uso de **useReducer**, **Custom Hooks** y el tipado estático con TypeScript, implementando operaciones CRUD sobre una lista de tareas. [Ver sitio](https://to-do-list-example.vercel.app/).

# Funcionalidades

- Agregar nuevas tareas.
- Editar tareas existentes.
- Eliminar tareas.
- Eliminar todas las tareas.
- Marcar tareas como completadas o pendientes.
- Contador de tareas totales.
- Contador de tareas completadas.
- Contador de tareas pendientes.
- Validación para impedir tareas vacías.
- Validación para impedir tareas duplicadas.
- Interfaz responsive.
